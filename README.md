## 🚀 Project Overview

This project transforms raw IMDb movie data into an interactive business intelligence dashboard using **Power BI**. It enables users to explore movie performance through dynamic filters and visualizations.

The dashboard helps answer questions such as:
- Which directors receive the most audience votes?
- How has movie revenue changed over the years?
- What is the average IMDb rating?
- How do runtime and revenue vary across movies?

---

## 📈 Dashboard Features

- 🎭 Filter movies by **Genre**
- 🎬 Filter by **Director**
- ⭐ Filter by **Actors**
- 📅 Filter by **Release Year**
- 📊 KPI Cards showing:
  - Average IMDb Rating
  - Average Votes
  - Average Revenue (Millions)
  - Average Runtime (Minutes)
- 📉 Revenue Trend by Year
- 🎥 Director-wise Average Votes
- 📦 Metascore Distribution by Year
- 🔄 Fully Interactive Dashboard

## 📌 Insights

- Revenue has generally increased over the years.
- Christopher Nolan has the highest average audience votes among the listed directors.
- Most movies have IMDb ratings between **6 and 8**.
- Average movie runtime is approximately **113 minutes**.
